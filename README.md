# Тестовое задание

Десктоп-приложение на WPF для управления сотрудниками, контрагентами и заказами.

## Что использовалось

- .NET 8
- NHibernate + FluentNHibernate
- MySQL / MariaDB
- Microsoft.Extensions.DependencyInjection

## Как запустить 

1. Установить MySQL / MariaDB
2. Склонировать репозиторий
3. Выполнить `db_init.sql` для создания бд
4. В `appsettings.json` указать свои параметры подключения.
5. В папке проекта `dotnet run --project TestTask`

## Какие особенности

1. Использован паттерн репозиторий
2. Использован DI контейнер
3. Работника/контрагента нельзя удалить при наличии активного заказа
4. Валидация на все поля ввода
5. Ленивая загрузка отключена, чтобы избежать N+1 проблемы

## Что можно добавить

1. Паттерн Unit of Work. Сейчас каждый метод репозитория открывает свое подключение, в то время как UoW может объединить несколько операции в сессию с одной транзакцией.
